# Social Media Agent

Scaffolded monorepo for an agentic AI YouTube publisher.

## LLM Provider
This project uses xAI Grok through an OpenAI-compatible endpoint:
- `XAI_API_KEY`
- `XAI_BASE_URL` (default: `https://api.x.ai/v1`)
- `XAI_MODEL` (default: `grok-3-mini`)

Do not commit real API keys. Add secrets in local `.env` only.
